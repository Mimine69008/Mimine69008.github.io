Situation Professionnelle 2

Déploiement d’un poste Windows et intégration au domaine.

Contexte

Un nouveau collaborateur arrivait dans l’entreprise et un poste devait être préparé selon les standards du service informatique. L’objectif était de déployer un poste Windows 10, l’intégrer au domaine Active Directory et appliquer les stratégies de sécurité.

Objectifs

  • Installer Windows 10 et effectuer les mises à jour
  • Rejoindre le domaine Active Directory
  • Configurer le profil utilisateur
  • Installer les logiciels métiers
  • Appliquer les GPO de sécurité
  • Tester le poste avant livraison

Travail réalisé

  • Installation propre de Windows 10
  • Configuration réseau + DNS du domaine
  • Intégration au domaine via AD
  • Installation des logiciels : Office, antivirus, outils internes
  • Application automatique des GPO (mot de passe, restrictions, bureau)
  • Tests finaux : connexion, accès aux partages, imprimantes
  • Remise du poste au collaborateur + documentation

Compétences validées

  • Déploiement de systèmes Windows
  • Gestion Active Directory
  • Installation et configuration logicielle
  • Application de stratégies GPO
  • Support utilisateur
  • Documentation technique
