Journal de stage

Résumé des tâches réalisées chaque semaine.

Semaine 1

  • Découverte de l’infrastructure réseau et systèmes
  • Présentation des outils internes : Zabbix, GLPI, Active Directory
  • Premiers tickets support (imprimantes, accès réseau, VPN)
  • Analyse des VLAN existants et de la topologie Cisco

Semaine 2

  • Déploiement de postes Windows 10/11
  • Intégration au domaine Active Directory
  • Installation de logiciels métiers
  • Configuration d’un VPN utilisateur + tests

Semaine 3

  • Supervision réseau via Zabbix (alertes CPU, RAM, ping)
  • Analyse de logs Windows et Linux
  • Durcissement de serveurs (GPO, SSH, firewall)
  • Documentation technique des procédures

Semaine 4

  • Maintenance réseau (switchs Cisco, ports, VLAN)
  • Résolution d’incidents complexes (DNS, DHCP, routage)
  • Tests de continuité de service
  • Bilan du stage avec l’équipe informatique
