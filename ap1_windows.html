Déploiement d’un serveur Windows

Mise en place d’un domaine Active Directory complet et gestion centralisée des utilisateurs.

Objectifs du projet

  • Installer Windows Server 2019
  • Configurer le rôle Active Directory Domain Services
  • Créer un domaine et organiser les OU
  • Gérer les utilisateurs et les groupes
  • Mettre en place des stratégies GPO
  • Sécuriser les partages et les accès

Compétences mobilisées

  • Installation et configuration de Windows Server
  • Déploiement d’un domaine Active Directory
  • Gestion des OU, utilisateurs et groupes
  • Création et application de GPO
  • Configuration des permissions NTFS et partages réseau
  • Intégration de postes clients au domaine

Travail réalisé

J’ai déployé un serveur Windows Server 2019 et mis en place un domaine complet permettant la gestion centralisée des utilisateurs et des ressources. Plusieurs postes clients ont été intégrés au domaine et des stratégies de sécurité ont été appliquées.

  • Installation du rôle AD DS et promotion du serveur en contrôleur de domaine
  • Création d’unités d’organisation (RH, Comptabilité, Direction…)
  • Gestion des comptes utilisateurs et groupes
  • GPO : mot de passe, restrictions, configuration du bureau
  • Partages sécurisés avec permissions NTFS + partage réseau
  • Tests de connexion et vérification des stratégies
